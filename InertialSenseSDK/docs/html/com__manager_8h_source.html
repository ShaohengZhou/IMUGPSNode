95 /*
96 Packet Overview
97 
98 Byte
99 0 Packet start byte
100 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
101 2 Packet counter (for ACK and retry)
102 3 Packet flags
103 
104 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
105 4-7 Data identifier
106 8-11 Data length
107 12-15 Data offset
108 16-19 Data start
109 (n-8)-(n-5) Last piece of data
110 // end data
111 
112 n-4 Reserved
113 n-3 Checksum high byte
114 n-2 Checksum low byte
115 n-1 Packet end byte
116 */
